# Git常用命令

#### 1.将当前文件夹初始化为git本地仓库

git init 

#### 2.设置全局用户名称

git config  --global user.name = "your name "

#### 3.设置全局用户邮箱

git config --blobal user.email "you email"

#### 4.下载项目

git clone https://github.com/rainxxx/doc.git

#### 5.查看状态

git status

#### 6.提交到暂存区

git add *
